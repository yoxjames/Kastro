interface TwilightPhase : SolarPhase

A Twilight Phase is a type of SolarPhase that includes only twilight phases. Phases like CivilTwilight, NauticalTwilight, AstronomicalTwilight would be TwilightPhases but SolarPhase.Day and SolarPhase.Night are not considered TwilightPhases.
