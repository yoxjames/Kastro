Intermediate

sealed interface Intermediate : LunarPhase

An intermediate moon phase. There are only 4 intermediate moon phases. Waxing Crescent, Waxing Gibbous, Waning Gibbous, Waning Crescent. These phases describe the state of the moon between the 4 Primary moon phases.
